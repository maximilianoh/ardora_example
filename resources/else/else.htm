4.3 Else

Consigna: Observa y prueba el uso de condicionales en Python
Objetivo: Explicar y ejemplificar la cláusula if-else como condicionales en Python.

# El condicional else permite ejecutar las partes de un código que no cumplen con las condiciones puestas en if, por eso va siempre luego de un if
##Código de ejemplo##
numero = 8
if(numero < 5):
 print("el numero es menor a 5")
else: # no cumple la condición del if
 print("el número es mayor a 5")

if(numero>2 and numero <7):
 print("es un número del 2 al 7")
else: # no cumple la condición del if
 print("el número no está en el rango de 2 a 7")
