2.2.1 Números

Consigna: Reconoce cada uno de los tipos de valores con sus respectivos ejemplos
Objetivo: Identificar y ejemplificar cada uno de los tipo de variables más utilizados en Python

Los números en Python se dividen entre enteros y con coma, pero la forma de asignarlos a una variable es igual. Para asignar un número en una variable se debe insertar el valor sin comillas
##Código de ejemplo##
# Números Enteros (int):
numero = 17
# Números con coma (float):
pi = 3.14159
print(numero, pi)
