2.2.2 Texto

Consigna: Reconoce cada uno de los tipos de valores con sus respectivos ejemplos
Objetivo: Identificar y ejemplificar cada uno de los tipo de variables más utilizados en Python

Los textos (también conocidos como string) se pueden guardar en una variable insertando el valor entre comillas dobles (""). Aunque lo que se ingrese sea un número o algo distinto a un texto o carácter, Python lo interpreta como texto y no como número o algo distinto, esto debe tenerse en cuenta cuando se hagan operaciones matemáticas.
##Código de ejemplo##
mensaje = "Hola Mundo"
mensaje = "111"
print(mensaje)
print(type(mensaje)) #es un texto (string), no un numero
