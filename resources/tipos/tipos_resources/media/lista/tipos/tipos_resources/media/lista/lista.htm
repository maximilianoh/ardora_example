2.2.4 Listas

Consigna: Reconoce cada uno de los tipos de valores con sus respectivos ejemplos
Objetivo: Identificar y ejemplificar cada uno de los tipo de variables más utilizados en Python

Sirve para agrupar múltiples valores en un mismo lugar (variable), por ejemplo para separar animales domésticos de salvajes, en donde podemos obtener cada valor individual indicando la posición entre corchetes, es decir valor_individual = lista[posicion]
Lo que hay que tener en consideración, es que por cómo se termina armando luego el código máquina las posiciones tienen un valor menos de lo esperado para la interpretación humana. Esto quiere decir que el elemento 1 de la lista para una persona, es el elemento 0 para una máquina, el elemento 2 de lista para una persona, es el elemento 1 para una persona, esto lo podemos probar en el siguiente ejemplo:
##Código de ejemplo##
a_hogar=["perro", "gato", "loro"]
a_salvajes=["tigre", "leon", "lobo"]
print(a_salvajes[0])
print(a_hogar[1])
print(a_salvajes[2])
# Se puede agregar elementos a lista
a_salvajes.append("puma")
# Se puede quitar elementos en la lista
a_hogar.remove("loro")
# Se puede saber cuántos elementos tiene en la lista
cantidad = len(a_salvajes)
print(a_salvajes, " tiene ", cantidad, " elementos")
print(a_hogar, " tiene ", len(a_hogar), " elementos")
