Consigna: Observa y prueba el uso de condicionales en Python
Objetivo: Explicar y ejemplificar la cláusula if-else como condicionales en Python.

# El condicional if permite ejecutar parte de un código siempre y cuando se cumplan las condiciones indicadas retornen True, por ejemplo podemos pensar en usar el paraguas sólo si llueve, eso se escribiría así:
##Código de ejemplo##
llueve = True
if(llueve): #si llueve es True y siempre se agregan los 2 puntos
 print("usar paragua") # siempre se deja un espacio/sangría (indentación) para mejor legibilidad
if(not llueve): # no cumple la condición
 print("no usar paragua") # no se muestra
