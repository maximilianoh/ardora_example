1.1 Python

Consigna: Lea la siguiente información sobre qué es Python
Objetivo: Introducir sobre el lenguaje de programación Python




Python es un lenguaje de alto nivel, pensado para ser relativamente sencillo de leer y escribir para las personas, y fácil de leer y procesar para las máquinas (lenguaje de bajo nivel). Hay dos tipos de programas que traducen lenguajes de alto nivel a lenguajes de bajo nivel: intérpretes y compiladores. Python es un intérprete, es decir, lee el código fuente de los programas tal y como ha sido escrito por el programador, lo analiza, e interpreta sus instrucciones sobre la marcha.
