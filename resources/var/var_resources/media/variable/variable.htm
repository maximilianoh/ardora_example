2.1.1 ¿Qué es una variable?

Consigna: Prueba el código de ejemplo para ver cómo funciona una variable
Objetivo: Definir qué es una variable

El valor es uno de los elementos fundamentales (como por ejemplo: una letra, texto o números) que manipula un programa. Una variable es un nombre que hace referencia a un valor.
La sentencia de asignación (=) crea nuevas variables y les asigna un valor, en este caso podemos ver como a la variable mensaje le asignamos como valor: Hola desde una variable.
##Código de ejemplo##
mensaje = "Hola desde una variable"
print(mensaje)
