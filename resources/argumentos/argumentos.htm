6.2 Parámetros

Consigna: Observa y prueba cómo añadir un argumento a una función en Python
Objetivo: Introducir y Mostrar el uso de argumentos en una función en Python

# Los parámetros son valores que se le pueden pasar a una función para que pueda realizar más acciones, por ejemplo si pensamos en una calculadora le podemos definir la función suma que recibe dos números
##Código de ejemplo##
def mostrar_suma(a,b): # se define a y b como parámetros de la suma
 print(a+b)
def devolver_suma(a,b):
 return a + b
mostrar_suma(10,15)
resultado = devolver_suma(17,12)
print(resultado)

# mostrar si un elemento está en una lista
def esta_en_la_lista(lista, elemento): # se define el parámetro lista y elemento
 encontrado = False
 for elem in lista:
  if elem == elemento:
   encontrado = True
   break # si encuentra el elemento que termine el bucle
 return encontrado

animales = ["perro", "gato", "tigre", "leon"]
print(esta_en_la_lista(animales,"leon"))
print(esta_en_la_lista(animales,"puma"))
