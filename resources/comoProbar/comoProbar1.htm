1.3 Cómo probar

Consigna: Observa la siguiente imagen sobre cómo probar el código que vayas generando a lo largo de este material complementario
Objetivo: Mostrar cómo se ejecuta el código realizado en esta plataforma



El código que aparece en la misma línea que un asterisco no es interpretado por Python y suele llamarse comentario, que sirve para hacer anotaciones o descripciones de lo que hace un código
