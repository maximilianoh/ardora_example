Consigna: Observa y prueba el uso de condicionales en Python
Objetivo: Explicar y ejemplificar la cláusula if-else como condicionales en Python.

# Muchas veces uno quiere hacer múltiples condicionales, por lo cual el if y el else no son son suficientes, para esos casos aparece elif, para crear tantas opciones como necesitemos. Su forma de escribir es exactamente igual al if
##Código de ejemplo##
numero = 8
if(numero <= 6):
 print("el numero es menor o igual a 6")
elif(numero == 7):
 print("el numero es 7")
elif(numero == 8):
 print("el numero es 8")
else: # no cumple la condición del if
 print("el número es mayor a 8")

if(numero>2 and numero <7):
 print("es un número del 2 al 7")
elif(numero>4 and numero <9):
 print("es un número del 4 al 9")
else: # no cumple la condición del if
 print("el número no está en ninguno de los rangos")
