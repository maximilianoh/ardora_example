5.1 While

Consigna: Identifica y prueba las formas de ejecutar una porción de código de forma repetida en Python
Objetivo: Reconocer y ejemplificar la utilización de bucles en Python

# los bucles son porciones de código que se ejecutan cierta cantidad de veces mientras su condición sea verdadera (True), es importante establecer una codición de corte para ello
##Código de ejemplo##
# Ejemplo de mostrar "hola" 10 veces
indice = 1
while(indice <11):
 print("hola")
 indice = indice + 1

# Ejemplo de mostrar elementos de una lista
lista = ["perro", "gato", "tigre", "leon"]
cantidad_elementos = len(lista) # 4 elementos
indice = 0
while(indice < cantidad_elementos):
 print(lista[indice])
 indice = indice + 1

# Ejemplo de cortar el bucle con break al encontrar la palabra tigre en la lista
lista = ["perro", "gato", "tigre", "leon"]
posicion = -1 # posicion no valida
indice = 0
while(True): # bucle infinito
 if lista[indice] == "tigre":
  posicion = indice # posicion 2
  break # se corta el bucle while
 indice += 1
print(posicion)
