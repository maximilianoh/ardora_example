# Las funciones son declaraciones de código que realizan ciertas operaciones que pueden ser llamadas múltiples veces, por ejemplo si pensamos en un sistema que saluda o despide al usuario
##Código de ejemplo##
def bienvenida(): # se define la función bienvenida
 print("Hola usuario")
def despedida(): # se define la función despedida
 print("Chau usuario")
bienvenida() # se llama a la función bienvenida
despedida() # se llama a la función despedida
despedida()
bienvenida()
