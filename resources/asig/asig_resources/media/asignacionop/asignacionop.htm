3.2.2 Asignación y Operación

Consigna: Observa y prueba el siguiente ejemplo sobre cómo se asigna y opera al mismo tiempo un valor a una variable
Objetivo: Reconocer cuando se asigna y opera un valor al mismo tiempo en una variable en Python

##Código de ejemplo##
# Asignar un valor
x = 7
# operar y asignar
x+=10
print(x) # suma lo que tenia x + 10
x-=13
print(x) # resta lo que tenia x -13
x*=5
print(x) # multiplica lo que tenia x * 5
