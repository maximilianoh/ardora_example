3.3.3 Menor y Mayor

Consigna: Identifica las formas de comparar valores que se tiene en Python
Objetivo: Ilustrar y resumir los distintos tipos de comparaciones de valores en Python

##Código de ejemplo##
# Si queremos saber si un número es mayor que otro
print(8>6)
print(6>8)
# Si queremos saber si un letra es mayor que otro
print("h">"y")
print("y">"h")
# Podemos incluir el extremo comparado si ponemos el igual luego
print(6>6)
print(6>=6)
# El símbolo de menor funciona de la misma manera
print(6<6)
print(6<=6)
print("h"<"y")
print(6<=8)
