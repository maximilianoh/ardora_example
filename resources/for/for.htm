5.2 For

Consigna: Identifica y prueba las formas de ejecutar una porción de código de forma repetida en Python
Objetivo: Reconocer y ejemplificar la utilización de bucles en Python

# Con este bucle se evita el problema del bucle infinito mal controlado (que sucede en el while), ya que la cantidad de veces que se ejecuta son limitadas
##Código de ejemplo##
# ejemplo recorrer una lista
lista = ["perro", "gato", "tigre", "leon"]
for animal in lista:
 print(animal)

# De la misma manera que en while, se puede cortar con break el flujo
lista = ["perro", "gato", "tigre", "leon"]
posicion = 0
for animal in lista:
 if animal == "tigre":
  break # se corta el bucle for
 posicion += 1
print(posicion)

# también se puede crear un bucle por número
for x in range(6): # de 0 a 5
 print(x)
for x in range(3,7): # de 3 a 6
 print(x)
